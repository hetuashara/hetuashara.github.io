Mac Apps

  • TextExpander - It expands typed pre-set abbreviations into bigger chunks of text. It works everywhere. EVERYWHERE. It really amazes me how people can live type without this utility.
  • Keyboard Maestro - This is the poster child of why mac is awesome. You need to check out the linked product page to explore the deep ocean that this app is.
  • Ulysses - Best long form writing app.
  • Nvalt - Best note taking app.
  • iStatus Menus - A menubar utility that shows you various stats. I use the network, CPU and memory monitors. Also, the date display as it has an inbuilt world clock.
  • Simplify - A menubar utility that can control iTunes, Spotify and Radio.
  • Tweetbot - A twitter client.
  • Sublime Text - My preferred code editor.
  • Airmail - The best mail client for mac. Believe me when I say best.
  • iTerm - A terminal alternative. Because I don't like original terminal’s looks.
  • Path Finder - A feature rich Finder alternative. If nothing else, the address bar at the top makes the app worth it.
  • Omnifocus - Everything that needs to be done goes here.
  • iSmartPhoto - After using tens of image viewers I have settled on this one. It is the best of the lot.
  • Moom - A nifty little utility that makes resizing windows hassle free.
  • Pushbullet - It helps me send stuff between mac and iphone.
  • Readkit - A powerhouse for RSS. I also read my Pocket saves there.
  • Speed Download - Now discontinued download manager app. I ahve tried all the current ones and this is still the fastest download accelerator.
  • MediaHuman Youtube Downloader - I hate waiting for 1080p videos to buffer on Youtube. I download videos via this app and watch them at my own leisurely time.
  • EiskaltDC ++ - My college has DC++. Yay. This is the only DC++ app which doesn’t destroy the fans on my mac when used for extensive periods of time.
  • Spotify - I really like the interface of the mac app.
  • iTunes - If you ask me, iTunes doesn’t deserve all the criticism it is getting. I find the library of music quite easy to manage and use.
  • Safari - 90% of browsing happens with this browser.
  • Chrome - FLASH.
  • Day one - A journaling app that you will actually use and like doing it.
  • BetterTouchTool - Unleash the full potential of your mac’s trackpad.
  • 1password - Because no one wants to remember all the passwords.
  • Fantastical - Best calendar app.
  • Yoink - A nifty app that helps better drag and drop.
  • gfxCardstatus - My mac has a faulty NVDIA Geforce card. It crashes my mac after certain time. The only solution is to use intel card. This app makes my mac force use the functioning intel graphics card.
  • Caffeine - Prevents mac from going to sleep.
  • Popclip - Select text and this app pops up. Then do stuff with that text.
  • Copyless - Best clipboard manager.
  • Adobe Premiere - Photo editing
  • Adobe Lightroom - Video editing
  • Audacity - Audio Editing
  • Dash - Offline reference for syntax of programming languages.
  • VLC - Playing videos.
  • Bartender - Keeps the menubar clean.

Browser Extensions of Safari

  • Speed Download - It sends all download links to the app. Also creates an often used contextual menu.
  • RSS menu - For some weird reason, some websites make it extremely hard to find the link for RSS feed. This extension comes in handy at that time.
  • 1password - Login to every website is one shortcut away.
  • Ghostery - Blocks annoying social media and advertisement-trackers.
  • Disconnect - Same as Ghostery. Double protection. FTW.
  • Adblock Plus - Blocks all kinds of ads. If you like you can also select specific elements of a website to be never shown.
  • Mailto - You can specify what to do when a email link is clicked. Safari opens a new mail box in the default mail app. Extremely frustrating.
  • Tablinks - Easy way to get Titles and URLs of all open tabs.
  • Youtube center - Customises Youtube.
  • DirectLinks - Shows clean tracking free links in browser status bar at the bottom.
  • LinkThing - Enhances control over how links are opened.
  • CustomReader - Helps customise Safari’s reader view.
  • Sessions - Extremely helpful to save those research sessions.
  • No more iTunes - Safari opens iTunes/Mac Appstore when visiting app pages. This saves you from annoyance.
  • Pinbar - An extension for using Pinboard better.
  • Pocket - An extension for saving pages to Pinboard in a faster way.
  • QuickStyle - Helps add custom CSS.
  • Font Face Ninja - Because sometimes you need to know the name of the font you are viewing.
  • Pushbullet - Sends and receives links from iphone.

Web Services

  • Spotify - Ideal for checking out new artists and albums.
  • Pocket - I have tried instapaper and readability on and off but always come back to pocket.
  • Pinboard - It works as a reservoir as well as a stream of links for me. Almost everything with a URL that interests me goes here. It is a paid service. Currently the subscription model has been changed to a per year basis but I purchased an account before the switch.
  • Dropbox - I use it exclusively to backup only my text documents.
  • Evernote - It is great to collect, store and archive stuff from around the web.
  • icloud - It syncs my calendar and tabs between iphone and mac. Nothing else.
  • Feedly - This is the only free option available right now in the market for using RSS. And it is pretty good.
  • Youtube - Surpringly high amount of usage - tutorials, entertainment, songs, etcetra.
  • Skype/Google Hangouts - For audio/video chat.
