Mac Apps

  • TextExpander - It expands typed pre-set abbreviations into bigger chunks of text. It works everywhere. EVERYWHERE. It really amazes me how people can type without something like this utility.
  • Keyboard Maestro - This is the poster child of why mac is awesome. You need to check out the linked product page to explore the deep ocean that this app is.
  • Byword - For the times when I want to focus on only one markdown article at a time.
  • CodeRunner - A no frills, inbuilt run environment code editor.
  • iStat Menus - A menubar utility that shows you various stats. I use the network, CPU and memory monitors. Also, the date display as it has an inbuilt world clock.
  • Simplify - A menubar utility that can control iTunes, Spotify and Radio.
  • Tweetbot - A twitter client.
  • Sublime Text - My preferred code editor.
  • Airmail - The best mail client for mac. Believe me when I say best.
  • Omnifocus - Everything that needs to be done goes here.
  • Day one - A beautiful journaling app.
  • iSmartPhoto - After using tens of image viewers I have settled on this one. It is the best of the lot.
  • Moom - A nifty little utility that makes resizing windows hassle free.
  • Reeder - A beautifully designed RSS reader.
  • Speed Download - Now discontinued download manager app. I have tried all the current ones and this is still the fastest download accelerator.
  • Downie - I hate waiting for 1080p videos to buffer on Youtube. I download videos via this app and watch them at my own leisurely time.
  • Spotify - I really like the interface of the mac app.
  • iTunes - If you ask me, iTunes doesn’t deserve all the criticism it is getting. I find the library of music quite easy to manage and use.And the cheap ₹120 per month Apple music subscription is sweet.
  • Safari - 90% of browsing happens with this browser.
  • Chrome - FLASH.
  • Day one - A journaling app that you will actually use and like doing it.
  • BetterTouchTool - Unleash the full potential of your mac’s trackpad.
  • 1password - Because no one wants to remember all the passwords.
  • Fantastical - Best calendar app.
  • Offline Pages Pro - For archiving websites and webpages.
  • Fetching.io - Records and offers full text search of my browsing history. And also saves webpages I visit.
  • Yoink - A nifty app that helps better drag and drop.
  • gfxCardstatus - My mac has a faulty NVDIA Geforce card. It crashes my mac after certain time. The only solution is to use intel card. This app makes my mac force use the functioning intel graphics card.
  • Popclip - Select text and this app pops up. Then do stuff with that text.
  • Copyless - Best clipboard manager.
  • Adobe Premiere - Video editing
  • Adobe Lightroom - Photo editing
  • Audacity - Audio Editing
  • Dash - Offline reference for syntax of programming languages.
  • VLC - Playing videos.
  • Bartender - Keeps the menubar clean.

Browser Extensions of Safari

  • 1password - Login to every website is one keyboard shortcut away.
  • Adblock - Blocks all kinds of ads. If you like you can also select specific elements of a website to be never shown.
  • Mailto - You can specify what to do when a email link is clicked. Safari opens a new mail in the default mail app. That is extremely frustrating.
  • Tablinks - Easy way to get Titles and URLs of all open tabs.
  • Youtube center - Customises Youtube.
  • DirectLinks - Shows clean tracking free links in browser status bar at the bottom.
  • LinkThing - Enhances control over how links are opened.
  • CustomReader - Helps customise Safari’s reader view.
  • No more iTunes - Safari opens iTunes/Mac Appstore when visiting iTunes/Appstore pages. This saves you from that annoyance.
  • Pocket - An extension for saving pages to Pocket in a faster way. Adds a contextual menu option that makes it super fast.
  • Reddit Enhancement Suite - Can do lots of tweaks to reddit. But by far the best one, for me, is hide child comments.
  • Fetching.io - Helps the mac app function.
  • Grammarly - Helps catch those silly mistakes while typing fast, and also those "oh I didn't realise that was a relative clause" type of mistakes.

iPhone Apps

  • Overcast - Smart speed in this app is the biggest thing to happen to podcast for me.

Web Services

  • Spotify - Ideal for checking out new artists and albums.
  • Pocket - I have tried instapaper and readability on and off but always come back to pocket.
  • Pinboard - It works as a reservoir as well as a stream of links for me. Almost everything with a URL that interests me goes here. It is a paid service. Currently the subscription model has been changed to a per year basis but I purchased an account before the switch.
  • Dropbox - I use it exclusively to backup only my text documents.
  • icloud - I use it to sync my calendar, contacts, keyboard shortcuts, and tabs between iphone and mac.
  • Feedly - This is the only free option available right now in the market for using RSS. And it is pretty good.
  • Telegram - For text messaging.
  • Youtube - Surpringly high amount of usage - tutorials, entertainment, songs, etcetra.
  • Apple Music
